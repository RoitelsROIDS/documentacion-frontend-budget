Firebase Authentication

Firebase Authentication proporciona una forma fácil de autenticar a los usuarios en la aplicación utilizando diferentes proveedores de autenticación, como correo electrónico/contraseña y Google.

Configuración e inicialización​

En primera instancia se establece la configuración de Firebase (API key, authDomain, etc.). Luego se inicializa la aplicación de Firebase utilizando la configuración proporcionada en el objeto firebaseConfig. Luego se obtiene una instancia del servicio de autenticación de Firebase utilizando getAuth(app). También se crea una instancia del proveedor de autenticación de Google utilizando new GoogleAuthProvider().

/src/services/firebase.tsx
const firebaseConfig = {
    // Configuración de Firebase (API key, authDomain, etc.)
    // Reemplazar con la configuración correspondiente a tu proyecto de Firebase
}

// Inicializar Firebase
const app = initializeApp(firebaseConfig)
const provider = new GoogleAuthProvider()

// Obtener una referencia al servicio de autenticación de Firebase
export const auth = getAuth(app)

info

Para obtener más información sobre la configuración del servicio: https://firebase.google.com/docs/web/learn-more#config-object

Inicio de sesión con correo electrónico y contraseña​

La función loginWithUserAndPass permite iniciar sesión con correo electrónico y contraseña. Recibe un objeto que contiene el correo electrónico y la contraseña del usuario. Utiliza el método signInWithEmailAndPassword del servicio de autenticación de Firebase para realizar la autenticación. Devuelve una promesa que se resuelve en true si el inicio de sesión es exitoso, o se rechaza en false si hay algún error.

/src/services/firebase.tsx
export const loginWithUserAndPass = ({ email, password }: { email: string; password: string }) => {
    const promise = new Promise((resolve, reject) => {
        signInWithEmailAndPassword(auth, email, password)
            .then((result) => {
                console.log(result)
                resolve(true)
            })
            .catch(() => {
                reject(false)
            })
    })
    return promise
}

tip

La función loginWithUserAndPass es importada en el componente que posee el formulario de inicio de sesión (LoginForm.tsx).

Cierre de sesión del usuario​

La función logout permite cerrar sesión del usuario actual. Utiliza el método signOut del servicio de autenticación de Firebase para cerrar sesión. Después de cerrar sesión, la página se recarga automáticamente para refrescar la aplicación.

/src/services/firebase.tsx
export const logout = async () => {
    await auth.signOut().then(() => {
        window.location.reload()
    })
}

tip

La función logout es importada en el componente que posee el botón de cierre de sesión (el componente Menu dentro de TopBar.tsx).
